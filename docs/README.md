# ArgusMonitor Docs

Self-hosted documentation site for ArgusMonitor using **Docusaurus**.

## Local development

```bash
cd docs
npm install
npm start
```

## Build

```bash
cd docs
npm run build
```

## Purpose

This docs site covers:
- product overview
- quick start
- hosted vs self-hosted
- pricing / AI credits / BYOK
- agent install + systemd deployment
- service discovery
- alerts
- AI copilot
- read-only host inspections

## Target

Intended deployment target:
- `docs.argusmonitor.com`
