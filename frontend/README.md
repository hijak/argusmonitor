# Vordr Frontend

React 18 + Vite + TypeScript frontend for Vordr.

## Run

```bash
npm install
npm run dev
```

The Vite dev server runs on `http://localhost:8080` and proxies `/api` requests to `http://localhost:8000`.

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
npm run test
```
